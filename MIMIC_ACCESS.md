# MIMIC-IV access boundary

MIMIC-IV v3.1 is credentialed data and is not redistributed by EHRDyn-ICU.
This minimal-runtime release contains no MIMIC rows, derived result tables,
checkpoints, split membership, or credentialed reconstruction implementation.

The public package supports installation, synthetic-fixture scoring, and the
constructed-environment entrant workflow. Reconstructing or evaluating the
real-EHR tasks requires a separately governed credentialed workflow and is
outside this release.

Official access information:

- MIMIC-IV v3.1: https://physionet.org/content/mimiciv/3.1/
- Credentialed DUA: https://physionet.org/content/mimiciv/view-dua/3.1/
- Training instructions: https://physionet.org/about/citi-course/
- Credentialing FAQ: https://physionet.org/about/faqs/

Do not upload MIMIC-IV rows or restricted derivatives to public repositories,
third-party APIs, shared prompts, or unapproved online services.
