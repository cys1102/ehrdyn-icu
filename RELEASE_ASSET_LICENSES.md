# Release asset redistribution

The v2.0.1 runtime source, schemas, documentation, task cards, synthetic
fixtures, and release manifests are covered by the repository MIT License.

The release excludes MIMIC-IV files, `public_bundle`, machine-readable
MIMIC-derived results, derived row-level interfaces, private runtime outputs,
model checkpoints, predictions, trajectories, and scientific result bundles.
MIMIC-IV remains subject to PhysioNet credentialing and is not redistributed.
Third-party dependencies retain their own licenses and are listed by the
frozen dependency lock.

The GitHub source archive and attached checksum/manifest assets contain only
repository-authored or mechanically generated public materials described
above.
