# EHRDyn-ICU v1.3.0 - Recursive Constructed Benchmark Entrant

This release adds a complete public prediction-to-policy entrant workflow over
all 40 constructed environments. It includes typed recursive point and
probabilistic schemas, uncertainty metrics, frozen H4 support-only planning,
full-episode direct return, repeated-dataset OPE, entrant examples, negative
fixtures, deterministic checks, and aggregate KDD235B evidence.

The included entrant is a conformance demonstration and is not a scientific
leaderboard submission. The release does not provide or validate EHR rows,
clinical treatment effects, policy benefit, or deployment readiness.
