# Frozen KDD235B full external entrant contract

KDD235B is re-executed from public KDD235A commit
`4739d59392a660cf215c29ebca02fb2f52cd7804` because the previously reported
terminal package was not recoverable from any local path or public Git ref.
No numerical result from that missing package is reconstructed or invented.

The run covers five profiles and eight environment seeds per profile. The
entrant is trained/initialized without final-role access, then evaluated on 11
common-origin recursive horizons. It exposes a native independent-Gaussian
distribution and an entrant-owned H4 support-only categorical-CEM policy.

Frozen inventory:

- 40 environment-specific checkpoint receipts;
- 440 forecast-horizon rows;
- 40 full-episode direct-return rows using 512 episodes each;
- 64 independent 256-episode logged datasets per environment;
- IS, WIS, CWPDIS, DR, WDR, and FQE under the primary cross-fitted denominator
  without clipping, yielding 240 environment-estimator summary rows.

The example entrant imports no benchmark transition class. Its deliberately
simple action-cost planner objective is an interface demonstration, not a
clinical reward model. It is excluded from the scientific leaderboard.

All exported results are aggregate. No EHR row, identifier, timestamp,
checkpoint state, trajectory, subject split, credential, or private path may
be released.
